## What this bibliography is

The bibliography is the reference infrastructure supporting the public release surfaces of the Panta Rhei Research Program.

It connects source literature, research influences, book references, result contexts, and verification-facing materials into a browsable reference corpus. It supports Publications, Program, Corpus, and Results surfaces without pretending to be a complete literature review.

The bibliography is not a casual footer page. It is a working public record of the scholarly horizon from which the program operates.

## Browse bibliography

Browse by domain group below — each entry links to a dedicated page showing the full citation, bibliographic metadata, and an editorial note on the reference's role in the research program.

Use the <a href="{{ '/bibliography/browse/' | relative_url }}">unified browse surface</a> for interactive exploration across the full 1,125-entry corpus. It lets you narrow the view by domain, role, entry type, citation status, book, and decade, and sort by year, title, or most-cited.

The raw BibTeX file (`references.bib`) is available for download for researchers who wish to use it directly.

## Citation and reuse notes

Use the bibliography as a reference aid, not as a substitute for citing the primary source. For program-level citation examples, see [Cite]({{ '/cite/' | relative_url }}). Original bibliography annotations are part of the site's CC BY 4.0 content unless a linked source states otherwise.
