## What this bibliography is

This is the shared reference corpus of the Panta Rhei Research Program. It contains the references cited across all seven books, organized by domain and annotated with short editorial notes explaining why each reference is included.

The bibliography is not a literature review. It is a working reference surface — a public record of the scholarly horizon from which the program operates.

## How to use this page

Browse by domain group below. Each entry links to a dedicated page showing the full citation, bibliographic metadata, and an editorial note on the reference's role in the research program.

The raw BibTeX file (`references.bib`) is available for download for researchers who wish to use it directly.
