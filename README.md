# 🌌 site

This repository is the **public delivery surface** of the **Panta Rhei Atlas** website.

It is intentionally thin and read-first.

## What lives here

- the public website delivery surface
- release-facing website content and deployment history
- stable public links into the Atlas

## What does **not** live here

- the canonical authoring source for Atlas content
- active collaborative drafting of framework pages
- experimental research materials

The canonical content/source brain lives in the private `atlas` repository.

## How this repo relates to the framework

The website is the **public atlas** of the framework.

It is not merely a marketing site or landing page.  
It is the public navigation surface through which the framework becomes explorable.
