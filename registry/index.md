## The Research Graph

The Registry is the canonical machine-readable substrate of the Panta Rhei research program. It tracks **4547 objects** — theorems, definitions, propositions, lemmas, axioms, remarks, and conjectures — across all seven books, with explicit dependency edges, scope labels, and formalization status.

## What each object page shows

Every registry object has its own page showing:
- **Identity**: canonical ID, type, name, scope, formalization status
- **Location**: book, part, chapter
- **Dependencies**: what this object depends on and what depends on it
- **Verification**: Lean module and symbol (if formalized)
- **Summary**: what the object states or defines

## Dashboards

Each book has a dashboard showing type distribution, formalization coverage, scope distribution, dependency depth, and the most central objects.
