Chapter [ch:goedel-and-the-vm-boundary] diagnosed
incompleteness and undecidability
as $E_2$ boundary phenomena.
Chapter [ch:proof-theory-as-e3] lifted the vantage point
to $E_3$, where the derivation system
becomes an object of study.
This chapter unifies both perspectives.
Cantor's diagonal argument, Russell's paradox,
G\"odel's incompleteness sentence, and Turing's halting problem
are diagnosed as four instances of a single phenomenon:
an $E_2$ object attempting an operation
that requires $E_3$ self-modelling.
Each paradox is the error message
produced when the boundary is crossed.
