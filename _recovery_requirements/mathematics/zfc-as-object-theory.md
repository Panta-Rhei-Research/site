# ZFC as Object Theory

## Requirement

The kernel should represent ZFC as a formal object-language without adopting ZFC as ambient ontology.

## What This Target Requires

- ZFC syntax, formulas, axioms, inference rules, and proof objects.
- Object-level derivability without automatic transfer of intended set-theoretic semantics.
