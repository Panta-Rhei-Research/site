The genetic code's 20 amino acids are mapped to BSD's rational carrier structure, with the codon table interpreted as a motivic L-function readout. Codon degeneracy—64 codons encoding 20 amino acids plus stop—is not waste but Shannon-type error correction. The central dogma (DNA → RNA → protein) is morphism composition in the code category, and DNA replication is blueprint propagation with fidelity ensured by proofreading and mismatch repair.
