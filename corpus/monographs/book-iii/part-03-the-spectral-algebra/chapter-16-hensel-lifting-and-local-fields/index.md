The CRT Decomposition Theorem supplied a finite factored ring at each primorial depth, but the decomposition was horizontal — one prime field per prime. The vertical direction — extending from ℤ/pℤ to ℤ/p²ℤ to ℤ/p³ℤ and so on — requires a lifting procedure. Classical Hensel lifting uses Newton iteration with a minus sign: aₙ₊₁ = aₙ − f(aₙ)/f′(aₙ). The τ-kernel has no minus sign to import, so this chapter delivers a complement-arithmetic version. If f(aₙ) ≡ 0 (mod pⁿ), write f(aₙ) = pⁿ·qₙ; the correction residue is cₙ = uₙ⁻¹·(p − qₙ) mod p where uₙ = f′(aₙ) mod p is a unit; the lifted root is aₙ₊₁ = aₙ + cₙ·pⁿ. The expression p − qₙ is the modular complement in {0,…,p−1}, not a negative number. The derivative uₙ is preserved as a unit at each step, so the iteration continues indefinitely without any analytic convergence argument. Assembling all lifts yields the τ-native p-adic integers ℤₚ^τ = lim← ℤ/pⁿℤ: the p-adic valuation is the D-coordinate of the NF address along the p-primary branch, and completeness — every eventually coherent thread extends uniquely — is an instance of Global Hartogs (I.T31) restricted to the p-primary sub-tower, with no metric or Cauchy sequence in sight.
