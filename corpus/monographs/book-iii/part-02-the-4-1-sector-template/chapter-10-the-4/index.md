The ABCD decomposition (*I.T04*) assigns each τ-object a unique quadruple Φ(x) = (A, B, C, D) where A = π(x), B = γ(x), C = η(x), D = α(x). Restricting this decomposition to the boundary character space Char(𝕃) — where every character χ_{(m,n)} is itself a τ-object — yields four sector projections P_A, P_B, P_C, P_D that are pairwise non-redundant: each tracks a structural feature of ρ (angular winding, exponent stratum, tetration stratum, radial depth) that is invisible to the other three at specific character families. This gives four primitive sectors, one per generator. The fifth generator ω is structurally different: its ABCD decomposition has P_B(ω) ≠ 0 and P_C(ω) ≠ 0 simultaneously, so it mediates coupling between the B- and C-sectors rather than defining a new independent direction. The 4+1 decomposition (*III.D13*) is therefore not postulated — it is induced by Langlands₀ from Chapter 9, and the Functoriality Theorem shows it commutes with F_E. A conjectural E₁ structural preview (*III.P03*) identifies the four primitive sectors with the four fundamental forces; no physical constants appear.

## What this chapter contributes

**Definitions / Axioms**
- *4+1 Sector Decomposition* (*III.D13*, τ-effective): Char(𝕃) ∖ {0} = S_D ⊔ S_A ⊔ S_B ⊔ S_C ⊔ S_ω. Four primitive sectors via dominant ABCD coordinates: S_D (P_D ≠ 0, no B-C mixing, generated by α, radial depth), S_A (P_A ≠ 0, no B-C mixing, generated by π, angular winding), S_B (P_B ≠ 0, P_C = 0, generated by γ, exponent stratum), S_C (P_C ≠ 0, P_B = 0, generated by η, tetration stratum). Each sector is exhaustively defined; sector membership is decidable at every finite primorial depth.
- *ω-Coupling Sector* (*III.D14*, τ-effective): S_ω = {χ ∈ Char(𝕃) | P_B(χ) ≠ 0 and P_C(χ) ≠ 0}. Characters with non-trivial projection on both B- and C-lobes simultaneously. The count is 4+1, not 5: the ω-sector does not add a new degree of freedom but correlates two existing ones. In layer-template terms, S_ω constitutes the predicate at E₁ (admissibility condition on primitive-sector data).
- *E₁ Physical Sector Preview* (*III.P03*, conjectural): structural identification at E₁ — D → gravity (base direction, Book V), A → weak force (angular winding / identity change, Book IV), B → electromagnetism (exponent stratum / multiplicative scaling, Book IV), C → strong force (tetration stratum / confinement growth, Book IV), ω → Higgs/mass mechanism (B-C crossing / chiral coupling, Book V). No physical constants, Lagrangians, or coupling values appear in Book III.

**Key results**
- *Non-redundancy of sector projections* (τ-effective): the four projections P_D, P_A, P_B, P_C are pairwise non-redundant — each carries information invisible to the others at specific character families. Follows from orbit uniqueness in the ABCD decomposition (*I.T04*).
- *Exhaustive decomposition* (τ-effective): the five sectors together with {0} exhaust Char(𝕃). Every non-trivial character belongs to exactly one sector. Proved by case analysis on (P_B(χ), P_C(χ)) with four mutually exclusive possibilities.
- *Functoriality of the 4+1 decomposition* (τ-effective): Φ(S_X) ⊆ Hol_X(τ³) for each X ∈ {D, A, B, C, ω}, and images are non-trivial. Follows from the Sector Preservation Theorem (*III.T05*) plus ABCD-preservation by τ-morphisms.
- *Enrichment compatibility* (τ-effective): the 4+1 decomposition commutes with F_E. The enrichment functor adds H_τ-module structure on top of existing sector data without altering sector membership. The decomposition at E₀ maps to the decomposition at E₁ under F_E.

**Notation**
