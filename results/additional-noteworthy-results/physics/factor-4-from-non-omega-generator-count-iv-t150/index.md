## Result summary

The factor 4 in m_H/m_n = (4 - X)/kappa(omega) equals the number of non-omega generators |{alpha, pi, gamma, eta}| = 4. Three independent structural derivations converge on the same value: (1) Generator count: |{alpha, pi, gamma, eta}| = 4 (most axiomatic; omega = gamma cap eta is excluded). (2) Lemniscate crossing channels: 2 lobes (left/right S^1) times 2 polarities (chi_+ / chi_-) = 4 crossing channel types. (3) Betti numbers: b_1(tau^3) + b_2(tau^3) - b_1(L) = 3 + 3 - 2 = 4, where tau^3 = tau^1 x_f T^2 (Kunneth: b_1 = 1+2 = 3, b_2 = 3 from T^2 fiber) and L = S^1 v S^1 has b_1 = 2.
