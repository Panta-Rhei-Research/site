## Source Excerpt

```lean
def laurent_sum (x k : TauIdx) : TauIdx :=
  go 1 k 0
where
  go (i remaining acc : Nat) : TauIdx :=
    if remaining = 0 then acc
    else go (i + 1) (remaining - 1) (acc + laurent_coeff x k i)
  termination_by remaining
```
