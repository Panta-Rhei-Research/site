Mathematical Content #

δ(x, y) = max { k : π_k(x) = π_k(y) } d(x, y) = 2^{-δ(x,y)}, d(x, x) = 0

Ultrametric: d(x,z) ≤ max(d(x,y), d(y,z)) Equivalently: δ(x,z) ≥ min(δ(x,y), δ(y,z))

Cylinder-ball correspondence: C_k(x) = { y : δ(x,y) ≥ k }

def Tau.BookII.Domains.disagree_depth (x y bound : Denotation.TauIdx) :
Denotation.TauIdx

[II.D12] First disagreement depth δ(x, y). Returns max { k : π_k(x) = π_k(y) } for k ≤ bound. If they agree for all k ≤ bound, returns bound + 1 (∞ proxy). Stage 0 always agrees (primorial 0 = 1), so δ ≥ 0.
