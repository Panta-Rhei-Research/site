## Source Excerpt

```lean
def hom_obj_identity_check (db : TauIdx) : Bool :=
  go 1 (db + 1)
where
  go (k fuel : Nat) : Bool :=
    if fuel = 0 then true
    else if k > db then true
    else
      let id_map := fun x => reduce x k
      is_reduce_compat_at id_map k && go (k + 1) (fuel - 1)
  termination_by fuel
```
