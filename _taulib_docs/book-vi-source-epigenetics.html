structure Tau.BookVI.Epigenetics.ChromatinPartition :
Type

[VI.D78] Chromatin Partition. Distinction clopen boundary at chromatin level: genome partitioned into euchromatin (active, D⁻¹(+)) and heterochromatin (silenced, D⁻¹(−)). The partition is a biological instance of VI.D04 (Distinction clopen boundary). Scope: τ-effective.

  • euchromatin_fraction : String

    Fraction of genome in euchromatin (active, open chromatin).

  • heterochromatin_fraction : String

    Fraction of genome in heterochromatin (silenced, condensed).

  • clopen : Bool

    Disjoint union covers entire genome (clopen property).

  • scope : String

    Scope: τ-effective (chromatin partition is a Distinction instance).
