structure Tau.Orbit.TauAutomorphism :
Type

A τ-automorphism: a bijection TauObj → TauObj that commutes with ρ.

  • toFun : Kernel.TauObj → Kernel.TauObj
  • invFun : Kernel.TauObj → Kernel.TauObj
  • left_inv (x : Kernel.TauObj) : self.invFun (self.toFun x) = x
  • right_inv (x : Kernel.TauObj) : self.toFun (self.invFun x) = x
  • rho_comm (x : Kernel.TauObj) : self.toFun (Kernel.rho x) = Kernel.rho (self.toFun x)
